# P4N4 Landing Page

Official landing page for **P4N4 :: Platform for Nexus Neural Network Nodes**, a unified developer platform for IoT and Edge AI engineers

![P4N4](https://img.shields.io/badge/P4N4-Edge%20AI%20Platform-ffd500?style=for-the-badge)
![License](https://img.shields.io/badge/license-MIT-00a8ff?style=for-the-badge)
![Status](https://img.shields.io/badge/status-active-ffd500?style=for-the-badge)

## Overview

This repository contains the source code for the P4N4 platform landing page, a modern, developer-focused marketing site built with pure HTML and CSS. The design emphasizes technical clarity and visual polish while avoiding generic marketing aesthetics.

**Live Demo:** [p4n4.com](https://p4n4.com)

## Style Guide

### Color Palette

```css
--bg-primary: #0a0e1a;      /* Deep blue-black background */
--bg-secondary: #121826;    /* Card backgrounds */
--bg-tertiary: #1a2332;     /* Elevated surfaces */
--text-primary: #e8edf5;    /* Primary text */
--text-secondary: #a0b0c8;  /* Secondary text */
--accent-primary: #ffd500;  /* Yellow - primary actions */
--accent-secondary: #00a8ff; /* Bright blue - highlights */
--accent-tertiary: #0066ff;  /* Deep blue - accents */
```

### Typography

- **Display Font**: [Barlow Condensed](https://fonts.google.com/specimen/Barlow+Condensed) (600, 700, 800)
- **Body Font**: [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) (400, 500, 700)

### Spacing System

- Base unit: `1rem` (16px)
- Section padding: `6rem` vertical
- Component gaps: `1.5rem` - `3rem`
